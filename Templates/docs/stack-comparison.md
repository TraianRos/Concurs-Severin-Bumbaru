# Comparatie Intre Stack-uri

## 01 MiniLibrary PHP

- Avantaje: setup simplu, OOP clar, usor de explicat la tabla.
- Dezavantaje: frontend mai putin spectaculos.
- Cand il alegi: cand vrei un CRUD rapid si stabil.

## 03 EventHub Flask

- Avantaje: cod scurt, separare buna pe straturi, SQLite simplu.
- Dezavantaje: ecosistemul Flask cere ceva disciplina in structura.
- Cand il alegi: cand vrei backend elegant si server-rendering clar.

## 02 SprintBoard React + Express

- Avantaje: separare moderna frontend/backend, bun pentru API-uri.
- Dezavantaje: mai mult setup si mai multe piese.
- Cand il alegi: cand tema cere o interfata mai interactiva.

## 04 RoomReserve Go

- Avantaje: structura explicita, performanta buna, control clar al serverului.
- Dezavantaje: mai mult cod boilerplate fata de Python/PHP.
- Cand il alegi: cand vrei sa iesi in evidenta fara sa complici prea mult UI-ul.

## 05 QuizLab FastAPI + WASM

- Avantaje: combinatie interesanta pentru demonstratie tehnica.
- Dezavantaje: necesita Emscripten si este cel mai greu de pregatit sub presiune.
- Cand il alegi: cand vrei un proiect de comparatie sau demonstratie, nu prima varianta de concurs.

## Recomandare

Pentru un hackathon scurt, incepe cu `01-mini-library-php` sau `03-event-hub-flask`. Daca ai nevoie de UI mai dinamic, treci la `02-sprint-board-react-express`.
